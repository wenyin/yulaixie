A tool for converting .twig  to .txt 
How To Use:
The k00000000001.twig is the file that needs to be converted and the 4444.txt is the converted file.
Unzip the file in the linux system such as Ubuntu or Fedora etc.
step1     cd  twig_dump/            ------------------------------------Enter the folder
step2     gcc twig_dump.c twig.c twig_file.c -o twig_dump---------Compile
step3     twig_dump  k00000000001.twig >4444.txt ----------------Convert
